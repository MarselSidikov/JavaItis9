# Рекурсия

* Стек - список, в который можно класть элементы в начало, и получать из начала с последюущим удалением.

```
{3, 5, 6, 7}

push(8)

{8, 3, 5, 6, 7}

pop()

{3, 5, 6, 7}

pop()

{5, 6, 7}
```

* Стек вызовов - определенная область памяти, куда помещаются вызовы процедур и функций

```
x(a) {
	return y(a + 1);
}

y(b) {
	return z(b * 2);
}

z(c) {
	return c / 3;
}

main() {
	x(5) * x(6);
}
```

* Формируем стек вызовов процедур/функций

```
	  V---------------------<4
	  |                      ^
pop() |-->z(6 * 2) = z(12)---- push()
      |	                     ^  
      |	                     |  
pop() --->y(5 + 1) = y(6)----- push()
 	  |	                     ^
      |	                     |
pop() |-->x(5)---------------- push()
      |                      ^ 
      V                      |
      --->main()-------------- push()

ИТОГО:

main() [4 * x(6)], то же самое для x(6).

	  V---------------------<4
	  |                      ^
pop() |-->z(7 * 2) = z(14)---- push()
      |	                     ^  
      |	                     |  
pop() --->y(6 + 1) = y(7)----- push()
 	  |	                     ^
      |	                     |
pop() |-->x(6)---------------- push()
      |                      ^ 
      V                      |
      --->main()-------------- push()


ИТОГО:

main() [4 * 4] -> 16
```

## Рекурсия

### Пример

```
n! = 1 * 2 * 3 * ... * n

5! = 1 * 2 * 3 * 4 * 5 = 120

6! = 1 * 2 * 3 * 4 * 5 * 6 = 720

6! = (1 * 2 * 3 * 4 * 5) * 6 = 5! * 6

n! = (n - 1)! * n
```
