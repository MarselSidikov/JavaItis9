# Задания

## Задание 1

* Реализовать программу следующего вида:

```JAVA
class Program {
	public static void main(String args[]) {
		int sum = 13 658;
		...
	}
}
```

```
sum - сумма, которую человек хочет снять в банкомате. Ваша задача вывести количество купюр каждого номинала для данной суммы.

5000 - 2
2000 - 1
1000 - 1

и т.д.
```
